Formularfelder in HTML

Es gibt verschiedene Arten von Formularsteuerelementen, mit denen man Informationen des Benutzers der Website einholen kann. Es gibt unteranderem Texteingabe Felder, Auswahlfelder und Sendeschaltflächen um die Formulare einzureichen. Eine solche Übertragung auf den Webserver funktioniert folgendermassen: 1. Der Benutzer trägt die Werte ins Formular ein, 2. Er klickt auf die Sendeschaltfläche, in diesem Moment werden die einzelnen Steuerelementen mit dem vom Benutzer eigegebenen oder ausgewählten Werten an den Webserver gesendet, 3. Der Webserver verarbeitet diese Informationen mit einer Programmiersprache wie PHP oder C# verarbeitet, unter Umständen werden die Werte in der Datenbank eingetragen, 4. Der Webserver leitet den Benutzer aufgrund der empfangenen Werte auf eine neue Seite weiter.

D.O.M

Ist die Abkürzung für Document Object Model und es ist eine Methode, welche den Aufbau aller HTML-Dokumente als Baumstruktur darstellt. Jeder "Ast" stellt somit zum Beispiel einen Absatz oder einen Titel dar. Für Browser bildet dies einen wichtigen Baustein für dynamische Webseiten. Ganz gut sehen kann man es im Chrome Developer Tool. Mit Windows öffnet man dies mit [F12], bei Linux mit [Ctrl]+[Shift]+[J] und mit Apple geht es mit [⌘]+[Optionstaste]+[J].

Was ist PHP?

PHP ist die Abkürzung für Personal Home Page Tools. Die Syntax von PHP ähnelt der Syntax von C oder pearl, damit dient bekanntlich zur Erstellung von dynamischen Webseiten und vielen anderen Möglichkeiten wie zum Beispiel das Eintragen von Daten in Datenbanken, aber bei uns im Kurs ging es grösstenteils um Formulare. PHP ist nicht kostenpflichtig, sondern es kann gratis auf einem Webserver verwendet werden. Wie in der Einleitung schon erwähnt ist PHP serverbasiert, das heisst, dass alle Prozesse auf dem Webserver ablaufen und somit die Clients weniger effektive Rechenleistung auf ihrem Gerät vorweisen müssen. Ursprünglich war PHP nur eine Ansammlung von Perl-Skripten, die Rasmus Lerdorf 1995 entwickelt hat, um seinen Online-Lebenslauf zu schreiben. Heute gilt dieser als Entwickler von PHP.

Aubau von PHP (Syntax)

als aller erstes sollte man bei einem PHP file wissen, dass man nicht nur php dort hinein schreiben kann, sonder auch andere sprachen wie html oder css können in einem php file vertreten sein. Wenn man aber ein php Skript starten will macht man dies mit:

<?php

wenn man das Skript wieder beenden will, benutz man folgenden tag:

?>

als nächstes machen wir ein Beispiel zu einer folständigen Funktion:

<?php 
function frame() { 
echo "Dies wird den Text ausgeben"; 
} 
?>

Um diese Funktion dann später zu benutzen, muss man folgende Syntax verwenden:

<?php 
frame(); 
?>

Validierung von PHP

Die Serverseitige Validierung ist eines der wichtigsten Themen, um seine Webseite / Server zu schützen. So können zum SQL-Injections verhindert werden mit nur einer Zeile PHP-Code, welche unten stehend gezeigt werden.


$prename = mysqli_real_escape_string($_REQUEST['prename']); Das $_REQUEST ist ein Platzhalter für POST oder GET, so müssen Sie sich nicht darauf achten, was Sie verwendet haben. Diese Funktion schliesst alle Strings aus, welche wie SQL-Befehle aussehen. Um die Seite vor Cross Site Scripting (XSS) zu schützen, können Sie die Variable wie folgt definieren:

$lastname = htmlspecialchars($_REQUEST['lastname']); Diese Funktion generiert aus allen Spezialzeichen wie > einen Zeichencode, sodass XSS-Attacken unschädlich gemacht werden. Diese Variablen könnten jetzt in eine Datenbank eingefügt werden, PHP könnte Sie jetzt aber auch weiterverarbeiten. Hier noch das Feld um den Code zu kopieren:

$prename = mysqli_real_escape_string($_REQUEST['prename']);
$prename = mysqli_real_escape_string($_REQUEST['prename']);

Javaskript (JS)

Von JavaScript(JS) hat sicher jeder schon mal gehört, jedoch sollte man JavaScript auf gar keinen Fall mit Java verwechseln, dies ist etwas komplett Anderes und darum soll es nicht gehen. Um wieder auf JavaScript zurückzukommen, JavaScript ist ebenfalls eine Skriptsprache und sie wurde ebenfalls in 1995 entwickelt. JavaScript hatte die Absicht Webseiten dynamischer zu machen, die selbe Aufgabe hat es auch heute noch. Das einzige was heute noch dazugekommen ist, ist, dass man JavaScript auch auf Servern und Microcontrollern vorfinden kann. JavaScript wurde früher hauptsächlich clientseitig eingesetzt. JavaScript ermöglicht aber beispielsweise mit dem Framework Node.js und als JScript bzw. JScript.NET auch serverseitige Anwendungen. Die Sprache wird auch als Skriptsprache für Spiele und Anwendungsprogramme eingesetzt. Dies ist weil der Sprachkern nur wenige Objekte enthält, deswegen kann das Skript relativ klein gehalten werden.

Warum sollte man JS lernen ?

diese Frage lässt sich leicht beantworten, denn es ist ganz einfach der Fall das alle drei Sprachen (HTML,CSS,JS) aufeinander aufbauen. Folgendes spricht dafür warum man Javascript als Web-Designer lernen sollte:

  1. HTML ist die Sprache des Text, heisst dass alles was man auf dieser Seite Lesen kann mit HTML gemacht wurde

  2. CSS ist die Sprache um den Text und das Design der webseiten zu formatieren

  3. Javascript ist die Sprache um Aktionen auf einer Webseite zu gestalten. es macht eine Webseite interaktiv. Javascript ist zusammengefasst da für zuständig, um zu sagen wie sich eine Seite verhalten soll.
