Entwicklungsumgebung

wenn man mit PHP arbeitet, gibt es keinen anderen Weg als diese php Dokumente auf einen Webserver zu laden um sie auszuführen. für HTML/CSS wird kein Server zum abrufen der Dateien Benötigt, da diese Clientseitig geladen werden deswegen benütigt man nicht mehr als ein Browser. Ich selbst benutze den chrome Browser von Google, da die Entwicklertools sehr ausgereift sind. Um mit HTML, CSS und PHP zu programmieren/schreiben braucht man nicht unbedingt einen speziellen Editor, jedoch erleichtert ein Editor die mühsame Fehlersuche wesentlich. Deswegen benutze ich auch den Visual Studio Code von Microsoft denn es können auch nützliche Add-Ons installiert werdem, welche beispielsweise die Tags automatisch schliessen. Sobald man den XAMPP Webserver heruntergeladen hat kann man mittels dem XAMPP Control Panel den Apache Dienst starten, und die Dashboard Seite von XAMPP über den localhost erreichen. Wenn man in der Browsersuchleiste localhost eingibt wird der htdocs Ordner nach Dateien durchsucht, welche er ausgeben kann. Im htdocs Ordner kann man weiter Unterordner erstellen und diese auch via "localhost/meinorder/datei.php" abrufen.
